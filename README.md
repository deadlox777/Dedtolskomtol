### 🚀 **Fitur yang Berfungsi Penuh**:
1. **WhatsApp Spam Report**  
   - Auto proxy rotation  
   - Custom jumlah report  
   - Anti-delay system  

2. **Download Video Sosmed**  
   - Support TikTok/Instagram/YouTube  
   - Simpan otomatis ke folder `Downloads`  

3. **Phone Number OSINT**  
   - Cek provider & lokasi  
   - Deteksi zona waktu  

4. **Device Monitor**  
   - Cek spesifikasi CPU & RAM  
   - Monitoring baterai (jika support)  

5. **Cek Umur Kartu SIM**  
   - Estimasi tahun aktifasi  
   - Tampilan tahun/bulan/hari  

### 📥 **Instalasi Mudah**:
```bash
pkg update && pkg upgrade
pkg install -y python git
git clone https://github.com/deadlox777/Dedtolskomtol
cd Dedtolskomtol
python Dedtolskomtol.py
```
